# Aether OS — Critical Mineral OS

**The "Trust Bridge" for rare earth supply chain compliance.**

A pitch-ready B2B SaaS prototype that proves ESG metrics, tracks molecular origins, and generates cryptographically secure Digital Battery Passports for the Caldeira Project REE supply chain.

## Quick Start

```bash
npm install
npm run dev     # → http://localhost:5173
npm run build   # production build
```

## 4 Views

| View | Audience | Purpose |
|------|----------|---------|
| **Operator** | Meteoric Resources | Plant efficiency, reagent use, FJH vs SX energy savings |
| **Regulator** | MPF / FEAM | Aquifer monitoring, sulfate/nitrate containment, UDC radiation, license timeline |
| **Compliance** | US DoD, EU Customs, Tesla | FEOC 0.00%, IRA compliance, EU Digital Battery Passport, blockchain ledger |
| **Executive** | C-suite, investors | NPV/IRR/CAPEX, 3-pillar ESG health, China vs Brazil supply map, green premium |

## Key Features

- **Live simulation** — 2s IoT telemetry pulse with realistic drift
- **Playback mode** — scrubable MREC batch journey from pit to magnet
- **Alert system** — threshold breach detection (pH, sulfate, radiation, aquifer)
- **PT-BR language toggle** — Regulator view in Portuguese for Brazilian agencies
- **Digital Twin Map** — interactive piezometer sensor map of PCAC region
- **Green Premium card** — spot NdPr vs certified price delta (+44%)
- **PDF Export** — print-to-PDF compliance report for EU DBP
- **Composite ESG score** — live 0–100 score across all three pillars
