# Caldeira / Brazil permitting glossary

Terms used in Aether OS map layers and executive copy. Definitions are operational summaries for UI context, not legal advice.

| Term | Meaning |
|------|---------|
| **ANM** | Agência Nacional de Mineração — national mining cadastre; authoritative for concession polygon geometry when exported from official systems. |
| **LP (Licença Prévia)** | Preliminary environmental license — first stage of Brazilian environmental licensing (viability). |
| **LI (Licença de Instalação)** | Installation license — required before construction. |
| **LO (Licença de Operação)** | Operating license. |
| **COPAM** | Conselho Estadual de Política Ambiental (MG) — state environmental policy council; votes on licenses. |
| **FEAM** | Fundação Estadual do Meio Ambiente (MG) — state environmental foundation; technical review. |
| **SUPRAM** | State environmental portal / process system for submissions (name may vary by state workflow). |
| **APA** | Área de Proteção Ambiental — protected area category. *APA Pedra Branca* is referenced in public filings and MPF commentary. |
| **CONGEAPA** | State council for APA management; buffer and use rules. |
| **IAC / IAD** | Ionic adsorption clay / ionic clay deposit — Caldeira’s geological style. |
| **TREO** | Total rare earth oxides — grade unit in resource tables. |
| **MREO** | Magnet rare earth oxides — Nd, Pr, Dy, Tb–weighted contribution in basket. |
| **PFS** | Pre-Feasibility Study — engineering/economics study; figures may be georeferenced for map layers with `confidence: georeferenced_figure`. |
| **JORC** | Australasian Code for Reporting of Exploration Results, Mineral Resources and Ore Reserves — governs public resource statements on ASX. |
| **UDC / INB** | Unidade de Descomissionamento de Caldas / Indústrias Nucleares do Brasil — legacy nuclear decommissioning complex; proximity cited in [MPF communications](https://www.mpf.mp.br/o-mpf/unidades/pr-mg/noticias/mpf-recomenda-a-retirada-de-pauta-de-votacao-de-processos-de-licenciamento-sobre-terras-raras-em-mg). |
| **MPF** | Ministério Público Federal — may issue recommendations; distinct from agency *approval* of LP. |
| **DD / AC / Auger** | Diamond drilling, air core (or rotary air blast), auger — collar types in exploration databases. |

When map footers say **non-survey**, polygons derived from deck figures or manual georeferencing are not cadastral substitutes.
